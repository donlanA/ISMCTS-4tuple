## Description
1. 4-tuple Server版
2. 把server資料夾放在4-tuple資料夾下編譯即可使用
3. 要把使用的data移進去server下面

## 資料夾架構示意：
> 4-tuple_1.3.7
  |> server (在這裡編譯)
    |> data
    |main.cpp
    |MyAI.cpp
    |MyAI.h
    |Tomorin.exe
  |DATA_impl.cpp
  |GST_impl.cpp
  |其他原本的檔案(pcg, main, header.h...)

## Modification
1. 修改成直接打包的模式
2. 把重複定義的define拿掉
